# IntelliJ IDEA New UI Theme

This Visual Studio Code theme pack aims to replicate the aesthetic of IntelliJ IDEA's new user interface. It features three distinct themes: a dark theme, a light theme, and a light theme with a light-colored header, providing options for various user preferences.
